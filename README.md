## Descrição do Projeto

O jogo consiste em desafiar o usuário a adivinhar um número secreto entre 1 e 100. A cada tentativa, o jogo fornece dicas informando se o número secreto é maior ou menor que o palpite fornecido.

## Funcionalidades

- [x] Gerar um número secreto aleatório.
- [x] Receber palpites do usuário.
- [x] Fornecer dicas com base nos palpites.
- [x] Informar quando o usuário acerta o número secreto.
- [x] Contabilizar o número de tentativas até o acerto.
